# Hei Introduction 更新日志

## v1.0b (2026-07-14)

### 新增功能

- **自定义页脚 HTML** - 后台可编写 HTML 代码渲染在首页页脚区域
- **网站配色方案** - 6 套预设渐变（天青 / 日落 / 深海 / 森林 / 薰衣草 / 暗夜）+ 完全自定义三色渐变，实时预览
- **自定义背景** - 三种模式：无背景 / API 随机图 / 上传图片；上传图片支持固定和轮播切换，可设置 3-60 秒间隔
- **捐款系统** - 完整的捐款页面，包含：
  - 扫码捐款（支付宝 / 微信 / QQ 钱包二维码上传）
  - 在线支付（易支付平台接入，支持支付宝 / 微信 / QQ 钱包跳转支付）
  - 可自定义页面标题、描述、预设金额
  - 首页捐款入口按钮（启用时显示）

### 改进

- **项目更名** - 项目名称定为"Hei Introduction"，版本 v1.0b
- **版权信息格式** - 页脚版权改为"网站名+年份+保留所有权利"
- **自述文件** - 完整重写 README.md，涵盖所有功能模块

### 移除

- **升级脚本** - 删除 up.php 及自动更新功能
- **备案号显示** - 前台页脚移除备案号展示

### 数据库结构（site_config 表）

| 字段 | 类型 | 说明 |
|------|------|------|
| custom_footer_html | LONGTEXT | 自定义页脚 HTML |
| color_scheme | LONGTEXT | 配色方案（JSON） |
| background_config | LONGTEXT | 背景配置（JSON） |
| donate_config | LONGTEXT | 捐款配置（JSON） |

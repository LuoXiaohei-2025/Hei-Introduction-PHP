# Hei Introduction

一个功能丰富的个人主页系统，集成个人信息展示、项目作品、技术文章、在线捐款等模块。采用现代化毛玻璃设计风格，注重用户体验与性能优化。

## 功能一览

### 核心展示

- **个人信息展示** - 头像、简介（Markdown 格式）、联系方式（QQ / 邮箱）
- **项目管理** - 展示个人项目，支持图标、链接和详细描述，模态框查看详情
- **文章系统** - Markdown 格式发布与管理，支持图片上传、实时预览、代码行号
- **打字机效果** - 动态欢迎语打字机动画
- **滚动动画** - 页面元素滚动入场动画效果

### 自定义外观

- **网站配色方案** - 支持 6 套预设渐变（天青 / 日落 / 深海 / 森林 / 薰衣草 / 暗夜）+ 完全自定义三色渐变，实时预览
- **自定义背景** - 三种模式：
  - 无背景（默认渐变）
  - API 随机图（输入随机图片 API 地址，每次访问获取不同背景）
  - 上传图片（支持多图上传、拖拽上传，可选固定或轮播切换，可设置 3-60 秒切换间隔）
- **自定义页脚 HTML** - 后台直接编写 HTML 代码渲染在页脚区域

### 捐款系统

- **扫码捐款** - 上传支付宝 / 微信 / QQ 钱包收款二维码
- **在线支付** - 接入易支付平台，支持支付宝 / 微信 / QQ 钱包自动跳转支付
- **预设金额** - 可自定义预设捐款金额按钮
- **自定义标题** - 可设置捐款页面标题和描述
- **首页入口** - 启用后首页自动显示捐款按钮

### 后台管理

- **仪表盘** - 当日访问 IP 数、近 7 日访问趋势柱状图
- **网站配置** - 网站名称、欢迎语、联系方式、页脚内容、配色方案、背景设置、捐款配置
- **项目管理** - 添加 / 编辑 / 删除项目，支持图标上传（自动转 WebP）
- **文章管理** - 发布 / 编辑 / 删除文章，内置 Markdown 编辑器 + 实时预览

### 其他特性

- **一键安装** - 三步安装向导（数据库 → 网站设置 → 管理员密码）
- **访问统计** - IP 去重统计，可视化趋势图表
- **图片优化** - 自动转换为 WebP 格式（GD 库），最大宽度 800px
- **响应式设计** - 完美适配电脑 / 平板 / 手机
- **零前端依赖** - 纯原生 HTML / CSS / JavaScript，无需 jQuery / Vue / React

## 技术栈

- **后端**：PHP 7.0+ / MySQL 5.6+ / PDO
- **前端**：HTML5 / CSS3 / 原生 JavaScript
- **图像处理**：GD 库（WebP 格式转换）
- **支付**：易支付标准接口（MD5 签名）

## 安装说明

### 环境要求

- PHP 7.0 或更高版本
- MySQL 5.6 或更高版本
- GD 库扩展（一般 PHP 已自带）
- Web 服务器（Apache / Nginx）
- 文件写入权限

### 安装步骤

1. 下载项目文件并上传到 Web 服务器根目录
2. 访问 `http://你的域名/install.php`
3. 按照安装向导完成三步配置：
   - 第一步：数据库配置（主机、数据库名、用户名、密码）
   - 第二步：网站设置（网站名称、欢迎语、头像）
   - 第三步：设置管理员密码
4. 安装完成后自动跳转到后台管理页面

### 安全建议

- 安装完成后删除 `install.php` 文件
- 设置强密码保护后台
- 定期备份数据库
- 保持 PHP 和 MySQL 版本更新

## 使用说明

### 后台管理

访问 `http://你的域名/admin.php` 进入后台，包含以下模块：

| 模块 | 功能 |
|------|------|
| 仪表盘 | 当日访问 IP 数、近 7 日访问趋势图表 |
| 网站配置 | 网站名称、欢迎语、联系方式、页脚内容、自定义页脚 HTML、配色方案、背景设置 |
| 项目管理 | 添加、编辑、删除项目，支持图标上传（自动转 WebP） |
| 文章管理 | 发布、编辑、删除文章，内置 Markdown 编辑器和实时预览 |
| 捐款配置 | 启用/禁用、页面标题描述、二维码上传、易支付接入 |

### 配色方案

在后台"网站基本配置"中选择配色方案：

- **预设渐变**：6 套内置方案，点击即用
- **自定义渐变**：3 个颜色选择器（背景浅色、主色调、强调色），实时预览渐变效果

### 自定义背景

在后台"网站基本配置"中设置背景：

- **API 随机图**：输入随机图片 API 地址
- **上传图片**：支持点击或拖拽上传多张图片，可选固定或轮播模式

### 捐款页面

在后台"捐款配置"中设置：

1. 开启捐款功能
2. 设置页面标题和描述
3. 上传支付宝 / 微信 / QQ 收款二维码
4. 配置易支付参数（接口地址、商户 PID、商户密钥）
5. 设置预设捐款金额

## 数据库结构

| 表名 | 用途 | 主要字段 |
|------|------|---------|
| site_config | 网站配置 | site_name, welcome_text, avatar_path, qq, email, footer_text, icp, custom_footer_html, color_scheme, background_config, donate_config, admin_password |
| projects | 项目管理 | name, icon_path, link, description |
| articles | 文章管理 | title, content, views |
| visits | 访问统计 | ip, visit_date, count |

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 安装失败 | 检查 PHP 版本、数据库连接信息、目录写入权限 |
| 图片上传失败 | 检查 uploads/ 和 picture/ 目录权限，确认 GD 库已启用 |
| 样式显示异常 | 清除浏览器缓存，检查 style.css 是否正确加载 |
| 数据库连接错误 | 检查 config.inc.php 配置，确认 MySQL 服务运行正常 |
| 易支付无法跳转 | 检查接口地址、PID、Key 是否正确，确认商户已审核通过 |
